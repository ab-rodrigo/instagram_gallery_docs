Overview

Enhance your e-commerce experience with image galleries powered by your Instagram posts. Add visual effects and captions to personalize your content. Use hashtags to filter media efficiently and create multiple galleries with unique layouts and settings. Images are updated automatically to keep your site fresh and engaging.

Features

  • Three types of visualization
  • Multiple galleries with different settings using the same API
  • Custom titles for each gallery
  • Hashtag-based filtering
  • Subtitles and captions
  • Pagination support
  • Image effects and transitions
  • Direct links to Instagram posts
  • Available in Portuguese and English

Requirements

  • Meta Facebook Developer Account — Facebook Developer Dashboard
  • Business account on Instagram — How to set up a business account
  • Access and modification permissions:
    • Extensions → Installer
    • Extensions → Extensions
    • Extensions → Modifications
    • Design → Layout

Installation

Follow these steps to install the Instagram Gallery extension in your OpenCart environment:

  1. Download the extension from the official OpenCart page and save the file to a known location on your computer.
  2. In your OpenCart admin panel, go to Extensions → Installer and click the Upload button.
  3. Select the previously downloaded file and click Open or double-click the file name. Wait for the upload and installation process to complete.
  4. Locate the extension in the Installed Extensions table and click the Install button to activate it.

Creating

Go to Extensions → Extensions, select Modules, find Instagram Gallery, and click the Add New button.

Integration

There are two types of galleries:

  • Parent Gallery: Used for API settings. You can create multiple parent galleries, each with its own configuration.
  • Child Gallery: Uses the API settings from a parent gallery. Multiple child galleries can be linked to one parent.

Parent gallery Select when creating a child gallery.

Display Settings

Configure how the gallery is displayed on the frontend.

  1. Navigate to the Design → Layouts section in your OpenCart admin panel.
  2. Select the layout where you want to add the Instagram Gallery module (e.g., Home, Category, Product).
  3. Click the Edit button for the chosen layout.
  4. In the layout configuration, find the section to add modules. Select Instagram Gallery from the dropdown menu.
  5. Choose the specific gallery you created earlier from the list of available galleries.
  6. Set the position where you want the gallery to appear (e.g., Content Top, Content Bottom, Column Left, Column Right).
  7. Specify the sort order if you have multiple modules in the same position.
  8. Click Save to apply the changes to your layout.

Facebook Developers / Instagram Business Accounts

Facebook Developers / Instagram Business Accounts

Before you start:

  1. Make sure you already have a Facebook Developer Account and that your store is not in maintenance mode.
  2. Ensure the Instagram account is a business account. See: How to Set Up an Instagram Business Account.

Creating Application

  1. Log in to your Facebook Developer Account and click My Apps.

  2. Click Create App.

  3. Enter the name of the application and click Next.

  4. Select Other and click Next.

  5. Select Business and click Next.

  6. Review the information and click Create App.

Application Setup

  1. Go to Add Product in the app dashboard, search for Instagram, and click Configure.

  2. In the side menu, go to App Functions > Functions and click Add People.

  3. Check Instagram Tester, add the account providing the media, and click Next.

  4. Log in to that Instagram account, go to More > Settings > Site permissions > Apps and websites. In the Tester Invitations tab, click Accept.

Access Token

  1. In the app dashboard, go to Instagram > API Setup with Instagram Login, find Generate Access Tokens, and click Generate Token.

  2. After logging in, click Allow.

  3. Check I Understand and click Copy.

  4. In the module settings under Integration, paste the token and click Validate Token.

  5. Check the Integration Status.
