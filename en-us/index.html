Overview

Enhance your e-commerce experience with image galleries based on your Instagram posts. Add effects and captions to give it a special touch. Use the hashtag feature to filter your media efficiently. Create different galleries for different layouts, each with personalized settings. Ensure that images are automatically updated to keep your content fresh and relevant.

Features

  • 3 types of visualization;
  • Multiple galleries with different settingss using the same API;
  • Title for each gallery;
  • Filter by#hashtag;
  • Subtitles;
  • Paginação;
  • Image effects;
  • Link to Instagram post;
  • Available in portuguese and english.

Requirements

  • Meta Facebook Developer Account. Facebook Developer Dashboard.
  • Business account on Instagram. How to set up a business account on Instagram
  • Access and modification permissions:
    • Extensions > Installer;
    • Extensions > Extensions;
    • Extensions > Modifications;
    • Design > Layout.

Support

Access the extension page and click Get Support.

Install

  1. Download the extension from the official page and save the file in a known location.

  2. Go to Extensions > Installer and click Upload button.

  3. Select the previously downloaded file, click Open or double-click the file name. Wait for the installation to complete.

  4. Locate the file in the Installed Extensions table and click Install button.

Integration

There are two types of galleries:

  1. Parent Gallery:

    This gallery will be dedicated to API settings. You can create multiple galleries of this type, each with its own API settings.

  2. Child gallery:

    This gallery type will use the Parent Gallery's API settings. You can create multiple galleries of this type for each Parent Gallery.

Parameter Description

Facebook Developers/Instagram Business Accounts

Facebook Developers/Instagram Business Accounts

Before you start:

  1. Make sure you already have a Facebook Developer Account and check that your store is not in maintenance mode.
  2. Make sure the Instagram account you are providing the images to is a business account. How to Set Up an Instagram Business Account.

Creating application

  1. Log in to your Facebook Developer Account, click on My Apps.

  2. Click Create App.

  3. Enter the name of the application and click Next.

  4. Select the Other option and click Next.

  5. Select the Business option and click Next.

  6. Review the information and click Create App.

Application Setup

  1. Go to Add Product in the app dashboard. Search for Instagram and click Configure.

  2. In the side menu, go to App Functions > Functions, click on Add people.

  3. Check the Instagram Tester option. Add the account that will provide the media and click Next.

  4. In the browser, log in to the Instagram account you entered in the previous step, go to More > Settings > Site permissions > Apps and websites. On the Tester Invitations tab, click Accept.

Access token

  1. In the application dashboard, on the left menu, go to Instagram > API Setup with Instagram Login, find the Generate access tokens and click Generate Token.

  2. After logging in, click Allow.

  3. Check the I Understand option and click Copy

  4. In the module settings, Integration tab, paste the token and click Validate token.

  5. Check the Integration status.

Displaying

  1. Go to Design > Layouts, choose the layout and click Edit
  2. Choose the position where you want to display the gallery and click Add module
  3. Select the gallery and click Save
