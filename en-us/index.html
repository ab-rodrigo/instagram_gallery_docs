Overview

Enhance your e-commerce experience with image galleries based on your Instagram posts. Add effects and captions to give it a special touch. Use the hashtag feature to filter your media efficiently. Create different galleries for different layouts, each with personalized settings. Ensure that images are automatically updated to keep your content fresh and relevant.

Features

  • 3 types of visualization.
  • Multiple galleries with different configurations using the same API.
  • Title for each gallery.
  • Filter by#hashtag.
  • Subtitles.
  • Paginação.
  • Image effects.
  • Link to Instagram post.
  • Available in Portuguese and English.

Requirements

  • Opencart 4.x with SSL certificate (required for Instagram API integration).
  • Download the extension in the version corresponding to your Opencart installation.
  • Meta Facebook Developer Account.Access the Facebook Developer Dashboard.
  • Business account on Instagram. How to set up a business account on Instagram
  • Access and modification permissions in Opencart:
    • Dashboard > Extensions > Installer.
    • Dashboard > Extensions > Extensions.
    • Panel > Extensions > Modifications.
    • Dashboard > Design > Layout

Support

Access the extension page and click Get Support.

Install

  1. Download the extension atofficial pageand save the file in a known location.

  2. From the Control Panel, go to Extensions > Instalador and click Upload.

  3. Locate the file, select it, and click Open or double-click the file name. Wait for the installation to complete. The extension will be listed in the Installed Extensions table.

  4. Click Install..

  5. Access the module settings in Extensions > Extensions, select Modules, locate the previously created gallery and click Edit.

Types

There are two ways to create a gallery:

  1. Main Gallery:

    This gallery will be dedicated to Instagram API configuration. You can create multiple Main Galleries, each with a different API configuration.

  2. Child gallery:

    This gallery type will use the API settings of the Parent Gallery. You can create multiple of this type for each Parent Gallery.

Creating

Click Extensions > Extensions from the Dashboard, select Modules, find Instagram Gallery and click Edit or New

Creating main gallery

Enter the name of the new gallery and click Salvar.

Creating child gallery

Enter the name of the new gallery, select the parent gallery and click the Save.

Note

There must be at least one main gallery with a configured and working API to create child galleries.

Configuration

General configurations

Parameter Description
Name Gallery name. Can be changed at any time. Duplication is not allowed.
Title Title to display above the gallery. Supports HTML and CSS.
Post Types Post types that will provide media for the gallery.
Album media type Types of media that will be searched within albums.
Album images Quantity of media to be searched within albums.
Update Refresh period.
Hashtag Used to filter posts that will provide media. Only posts that have the hashtag will be processed by the module. Posts without the hashtag will be ignored.
Status Enables the module.

View Settings

Parameter Description
Media by view Number of images that will be displayed.
View type Gallery style.
Pagination Displays pagination marks.
Zoom effect Choose the zoom effect that will be applied to the image when the user hovers their mouse over it.
Rounded edges Add rounded edges to images.
Inclinação Add tilt effect to images.
Color effect Choose the type of color effect that will be applied to the image when the user hovers over it.
Caption Display part of the post caption.
Link to original post Turn each image into a link that points to the corresponding Instagram post.
Mobile devices Enables gallery display on mobile devices with small screens (smaller than 576px).

Displaying

  1. Go to Design > Layouts, choose the layout you want to display the gallery in, and click Edit.

  2. In the chosen position, click Add module.

  3. Select the gallery and click Save.

Facebook Developers/Instagram Business Accounts

Facebook Developers/Instagram Business Accounts

Before you start:

  1. Make sure you already have a Facebook Developer Account and check that your store is not in maintenance mode.

  2. Make sure the Instagram account you are providing the images to is a business account. How to Set Up an Instagram Business Account.

Creating application

  1. Log in to your Facebook Developer Account, click on My Apps.

  2. Click Create App.

  3. Enter the name of the application and click Next.

  4. Select the Other option and click Next.

  5. Select the Business option and click Next.

  6. Review the information and click Create App.

Configuring the application

  1. Go to Add Product in the app dashboard. Search for Instagram and click Configure.

  2. In the side menu, go to App Functions > Functions, click on Add people.

  3. Check the Instagram Tester option. Add the account that will provide the media and click Next.

  4. In the browser, log in to the Instagram account you entered in the previous step, go to More > Configuration > Site permissions > Apps and websites. On the Tester Invitations tab, click Accept.

  5. Access the module settings in Extensions > Extensions, select Modules in Extension Type, locate the gallery you created earlier, and click Edit.

    In the API tab, copy the Store URL.

  6. Return to the application dashboard, in the left menu go to Instagram > API Settings with Instagram Login, find the item Configure Instagram Business Login and click Configure.

    Paste the Store URL and click Save.

  7. Still on the app dashboard, copy the Instagram App ID and Instagram App Secret Key.

  8. Return to the module settings, in the API tab, paste the Instagram App ID and Instagram App Secret Key into the corresponding fields. Click the Get Authorization.

  9. A pop-up will open requesting authorization, click Allow. If you are not logged into Instagram, you will need to enter your username and password before proceeding with the authorization.

  10. A confirmation screen will appear at the end. Return to the module settings and check the API Status. If necessary, refresh the screen.
