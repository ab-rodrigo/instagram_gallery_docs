Welcome

A complete step-by-step guide to installing, configuring and connecting the Instagram Gallery to your store.

Overview

Transform your Instagram feed into a dynamic storefront directly on your online store.

Increase customer interactivity and engagement in real-time by integrating an elegant and fully responsive gallery into your OpenCart pages.

Requirements

  • Access to OpenCart Extension Installer
  • Permission to manage system Layouts
  • Active Instagram account (Business)
  • Access to Facebook Developers Panel

Features

Essential

Three Display Types

Choose between different modern visual formats to present your Instagram photos in the way that best adapts to your theme design.

Flexibility

Multiple Galleries

Create and configure **multiple independent galleries** using the same API integration, applying distinct visual rules and behaviors to each.

Engagement

Direct Links

Redirect your visitors straight to the original posts on Instagram when clicking the photos, driving your likes and comments.

Custom Titles

Define exclusive titles and headers for each installed gallery block, matching the call to action to the page context.

Hashtag Filter

Surgically select and display only media containing specific terms, intelligently segmenting your feed content.

Fluid Visuals

Display captions and descriptions accompanied by elegant transition effects and hover behavior on images.

Installation

Follow the steps below to integrate the module into your OpenCart system securely.

01

Download

Get the official module directly from the OpenCart marketplace:

Official Page
02

Upload

In the store admin panel, go to Extensions > Installer. Click Upload and select the downloaded file.

03

Activation

Locate the module in the list of installed extensions and click the Install icon to activate it.

Important Notice about Cache:

OC 3.x Go to Extensions > Modifications and click the blue Refresh button to renew the system cache.
OC 4.x On the dashboard homepage, click the blue gear button in the upper-right corner and clear the Theme and SASS cache.

Accessing Settings

After installation, follow this workflow to configure your automation.

01

Side Menu

In the OpenCart admin panel, go to Extensions > Extensions.

02

Filter

Select the extension type as Modules and locate Instagram Gallery.

03

Action

Click Add New to create a new gallery, or Edit on the desired gallery to modify it.

1. General Configuration

Parameter Function
Name Gallery name displayed in the layout configuration.
Title Gallery title displayed in the layout configuration.
Update Image refresh interval period.
Status Enables the module.

2. View

Parameter Description
Image limit Defines the total number of images displayed in the gallery.
Post types Selects the publication sources that will feed the gallery.
Album media type Filters the type of media imported from albums.
Album images Limits the number of media items imported per album.
Hashtag Displays only posts containing the specified hashtag.
View type Changes the visual style of the gallery.
Pagination Displays the pagination controls.
Zoom effect Enables a zoom-in effect when hovering over the image.
Rounded borders Rounds the corners of the gallery images.
Tilt effect Slightly tilts the image on hover.
Color effect Changes the image tone on hover.
Caption Displays part of the post caption.
Link to original post Redirects to the original post on Instagram when clicking the image.
Mobile devices Optimizes the gallery display for smartphone screens (smaller than 576px).

3. Integration

Gallery types:

  • Parent Gallery: used for API settings. You can create multiple parent galleries, each with its own configuration.
  • Child Gallery: uses the API settings of a parent gallery. Multiple child galleries can be linked to a single parent gallery.

Parent gallery Select the source gallery when creating a child gallery.

Instagram API Integration

Before you start:

  • Make sure you have a Facebook Developer Account.
  • Verify that the Instagram account is a business account. Learn how to configure a business account on Instagram.

1. Creating the application

  1. Access your Facebook Developer Account and click on My Apps.

  2. Click on the Create App button.

  3. Enter the application name and click Next.

  4. Select the Other option and click Next.

  5. Check the Business use case and click Next.

  6. Review the information carefully and click Create app.

2. Application configuration

  1. In the side panel of the application, go to Add Product, locate Instagram and click on Set Up.

  2. In the left menu, navigate to App roles > Roles and click on Add People.

  3. Check the Instagram Tester option, type the account that will provide the gallery media and click Add.

  4. Log in to the Instagram account provided in the previous step, go to More > Settings > Website permissions > Apps and websites. In the Tester Invites tab, click Accept.

3. Access token

  1. Back in the Facebook dashboard, go to Instagram > API Configuration with Instagram Login, locate the Generate Access Tokens section and click on Generate Token.

  2. A window will request access authorization for the account. After logging in, click Allow.

  3. Check the I understand checkbox and click Copy.

  4. In your module settings, access the Integration tab, paste the generated code into the token field and click Save. Reload the page and check the Status panel to confirm the success of the operation.

Gallery Display

Configure on which pages and positions of your store the Instagram gallery will be displayed.

01

Access

In the side menu of your OpenCart admin panel, navigate to Design > Layouts.

02

Selection

Choose the layout of the page where you want to position the module (such as Home or Account) and click the blue Edit icon.

03

Positioning

Choose the desired position (top, footer, or columns), add the Instagram Gallery extension in the selector, and set the sort order.

04

Save

Click the Save button in the upper-right corner to apply the gallery to your store.

Layout Tip: If the module does not appear immediately on the storefront, make sure that the chosen position in the selected layout is active and rendered in your store's current theme.

Uninstallation

Follow the steps below for a clean and secure uninstallation.

01

Disable

Go to Extensions > Extensions, filter by Modules, locate the module, and click Uninstall.

02

Uninstall

Locate the module in the list of installed extensions and click the Uninstall icon.

03

Delete

Still in the list of installed extensions, click Delete.

Important Notice about Cache:

OC 3.x Go to Extensions > Modifications and click the blue Refresh button to rebuild the system cache.
OC 4.x On the dashboard homepage, click the blue gear button in the upper-right corner and clear both the Theme and SASS caches.
